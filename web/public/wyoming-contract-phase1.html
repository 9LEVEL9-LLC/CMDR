Phase 1: Foundation Platform - Scope of Work

This SOW is issued pursuant to the terms and conditions of the Master Services Agreement between 9LEVEL9 LLC Series 6 ("Company") and Wyoming Asset Protection Trust / Meuli & Associates ("Client").

Phase 1 Scope Of Work (Exhibit A)

This Statement of Work ("SOW") outlines Phase 1 deliverables for the Foundation Platform - a custom legal technology platform with client portal, matter management CRM, document assembly, compliance monitoring, content management system, and API integrations for Wyoming Asset Protection Trust legal practice.

Note: Phase 2 (AI Automation including Intelligent Intake, Email Automation, AI Oracle Agent, Automated Corporate Minutes, Advanced SMS/Voice, and Analytics) will be scoped and priced separately after Phase 1 is deployed and stable.

Summary

Wyoming Asset Protection Trust (WAPA) requires a custom technology platform to scale operations while maintaining premium client service. This Phase 1 deployment establishes the foundation: client portal with CompanySage API integration, matter management CRM, document assembly automation, Legal Operations Center compliance dashboard, content management system with 50,000 words of SEO content, and core API integrations.

Phase 1 delivers immediate value by controlling client experience, automating document workflows, monitoring compliance deadlines, and establishing strong online presence. Phase 2 will add advanced AI automation for intelligent intake, proactive monitoring, and sophisticated communication sequences once the platform is stable.

Phase 1 Project Deliverables

1. Client Portal & Dashboard

  • Vue.js responsive frontend with WAPA branding
  • Real-time CompanySage API synchronization for entity data
  • Entity dashboard, document vault, billing transparency
  • Service add-ons and appointment scheduling
  • Secure client messaging system

2. Legal Operations Center (LOC)

  • NOC-style compliance monitoring dashboard
  • Red/Yellow/Green status indicators for all entities
  • Expiration tracking (entities, RA renewals, leases, phone numbers)
  • Daily automated Secretary of State database monitoring
  • Proactive alert system (60/30/7 day warnings)

3. Matter Management CRM

  • Client lifecycle tracking (Lead → Active → Maintenance)
  • Matter-based organization for trusts and entities
  • Automated workflow checklists by matter type
  • Task assignments, deadline tracking, internal notes
  • LawPay billing integration for matter-based billing
  • Revenue and matter status reporting dashboards

4. Document Assembly & Template System

  • Handlebars template conversion for all legal documents
  • Multi-document generation (Trust + Holding LLC + Operating LLC)
  • Consistent variable mapping and naming across documents
  • Validation, error checking, version control
  • PDF generation with e-signature workflow integration

5. Content Management System (Strapi)

  • Self-hosted Strapi headless CMS installation
  • Landing page builder with service page templates
  • Blog/article publishing system with SEO optimization
  • Lead capture forms integrated with CRM
  • Vue.js component library integration

6. API Integration Layer

  • CompanySage API integration (entity data, RA status, documents)
  • Wyoming LLC Attorney API integration (lead tracking)
  • LawPay payment processing integration
  • Twilio SMS integration for basic notifications
  • Calendly scheduling integration
  • Webhook listeners for real-time updates

7. Infrastructure, Security & Deployment

  • Render or NuxtHub hosting environment (staging + production)
  • Database setup (PostgreSQL, Cloudflare D1, or client's preferred cloud database) with automated daily backups
  • GitHub organization and repository setup
  • Google Identity Platform OAuth2 authentication
  • SSL/TLS encryption, RBAC, activity logging
  • Application monitoring and error tracking

8. Content Creation (50,000 Words)

  • 30-40 comprehensive SEO-optimized articles
  • Human-written content (not AI-generated)
  • Wyoming asset protection trust topics
  • LLC formation and structure education
  • Compliance requirements and best practices
  • Published directly to Strapi CMS

9. Documentation, Training & Support

  • Technical architecture and API documentation
  • User guides for client portal and staff workflows
  • Video training sessions for platform usage
  • 30 days post-launch support for bug fixes and optimization

Phase 1 Development Timeline

Total Duration: 8 weeks from contract signing to production launch

  • Weeks 1-2: Foundation (database, APIs, GitHub, Strapi, hosting)
  • Weeks 3-4: Core Platform (client portal, CRM, documents, CompanySage API)
  • Weeks 5-6: Integration & Content (LawPay, LOC dashboard, content creation begins)
  • Week 7: Testing & Refinement (end-to-end testing, performance optimization)
  • Week 8: Launch & Training (deployment, staff training, documentation handoff)
